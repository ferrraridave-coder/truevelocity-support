1. Description of the App

True Velocity is a GPS-based speedometer application for iPhone. It displays real-time speed using your device’s built-in GPS, along with optional filtering and responsiveness settings.

Key features include:

  • Analogue speed dial
  • Large digital speed display
  • MPH and KMH unit selection
  • Adjustable speed response modes (Smooth, Balanced, Fast)
  • Adaptive filtering
  • Real-time GPS accuracy display

The App works without requiring an internet connection. Speed calculations are performed locally on your device.
